# web_app

Use this archetype for browser-based applications, dashboards, or thin clients.

## Good fit
- React, Next.js, Vite, or other browser-first projects
- web tools that may benefit from Codespaces or devcontainers

## Default posture
- mode: `standard`
- consider enabling Codespaces only when it genuinely reduces setup friction

## Recommended modules
- `release_management`
- `diagnostics`
- `qa_checks`
- `build_identity`
