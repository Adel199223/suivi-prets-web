# cli_tool

Use this archetype for small command-line utilities and automations.

## Good fit
- single-purpose helpers
- scripts that need clear usage docs but not a heavy governance surface

## Default posture
- mode: `lite`
- keep the harness lean unless the repo grows integrations or release needs

## Recommended modules
- `base_docs`
- `session_resume`
- `diagnostics`
- `qa_checks`
