# Guide Dette

## Use This Guide When

Use this guide when someone needs help adding a debt, logging a payment, adding an extra advance, or reading the remaining balance.

## Do Not Use This Guide For

Do not use this guide for parser logic, duplicate signatures, or storage migrations.

## For Agents: Support Interaction Contract

Guide the user through the flow in order: borrower, debt, payment or advance, then verification of the updated balance.

## Canonical Deference Rule

Canonical technical docs remain the source of truth if this guide becomes stale.

## Quick Start (No Technical Background)

1. Open the borrower page.
2. Choose the debt you want to update.
3. Click `Enregistrer un paiement` or `Ajouter une avance`.
4. Fill the amount and date if you have it.
5. Verify that `Reste a encaisser` changed as expected.
6. Open `Informations de la dette` to update the debt label or notes if needed.
7. From the timeline, use `Modifier` to correct an existing line (amount/date/detail) or `Supprimer` to remove one.
8. Use `Supprimer cette dette` in this view when you want the full debt and its lines removed.

## Terms in Plain English

- Paiement: money received back from the borrower.
- Avance: extra money lent after the original debt.
- Solde initial: the amount already owed when the debt record starts.
- Reste a encaisser: what still has to be paid back.
